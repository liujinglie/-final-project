# Linear Regression을 이용한 항공권 가격 예측

## 프로젝트 개요
본 프로젝트는 Python을 사용하여 항공권 가격을 예측하는 Linear Regression 모델을 구현한 것이다.
항공권의 기본적인 특성 정보를 입력으로 사용하여 가격을 예측함으로써,
수업 초반에 학습한 회귀 분석 기법을 실제 데이터에 적용하는 것을 목표로 한다.

## 데이터셋
본 프로젝트에서 사용한 데이터는 공개된 항공권 가격 데이터셋이며,
교육 및 학습 목적을 위해 사용되었다.

데이터에는 다음과 같은 항목이 포함될 수 있다.
- 출발지 / 도착지
- 항공사
- 출발 날짜 및 시간
- 경유 여부
- 항공권 가격 (예측 대상 변수)

## 모델
항공권 가격과 여러 항공편 특성 간의 관계를 분석하기 위해
scikit-learn의 Linear Regression 모델을 사용하였다.
해당 모델은 구조가 단순하고 해석이 용이하여 수업 내용과 잘 부합한다.

## 결과
학습된 모델은 입력된 항공편 정보에 따라 항공권 가격을 예측할 수 있으며,
실제 가격과 예측 가격 간의 차이를 통해 모델의 성능을 확인하였다.
모델 성능 평가는 Mean Absolute Error(MAE)를 기준으로 수행하였다.
